# Steganography in Wav audio using LSB

2 Режима: Encode и Decode

##Encode mode

В первое поле вводится полный путь .wav файла в который будет зашифированно сообщение (исходный файл никак не изменяется)

В второе поле вводится полный путь .wav файла который будет зашифирован (выходной файл с зашифрованным сообщением)

В поле с сообщением вводим сообщение, которое будет зашифровано. 
''' ВНИМАНИЕ! Поддерживается только ЛАТИНИЦА. С кириллицей будет выдавать ошибку.

После введения данных нажимаем кнопку старт и получаем путь нашего .wav файла с зашифрованной информацией


##Decode mode

В поле вводится полный путь .wav файла в котором есть зашифированное сообщение

После введения данных нажимаем кнопку старт и получаем сообщение, содержащееся в .wav файле.
